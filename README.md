Light Finder är en webbaserad applikation som hjälper dig hitta de mörkaste platserna nära dig för optimal stjärnskådning. Genom att analysera satellitdata (VIIRS DNB) identifierar applikationen områden med låg ljusförorening och presenterar de bästa observationspunkterna på en interaktiv karta.

Datasetet är för stort för GitHub och inkluderas därför inte i repot.
Det går dock att ladda ner via denna länk https://eogdata.mines.edu/nighttime_light/annual/v22/2023/
Placera filen i: /backend/data/data.tif
